Usage:
    ant init            % for initializing directories
    ant compile         % for compiling
    ant javadoc         % for generating javadoc
    ant dist            % for generating executable jar file
    ant -Darg0=source_dir_name -Darg1=destination_dir_name -Darg2=extension_name   % for running the program, leave extension_name blank for default
    ant copy            % for copying the doc file
    ant clean           % delete the output files
